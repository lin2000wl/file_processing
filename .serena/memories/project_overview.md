# MonkeyOCR 项目概述

## 项目目的
MonkeyOCR是一个基于结构-识别-关系（SRR）三元组范式的文档解析项目。它采用深度学习技术对PDF文档和图像进行OCR（光学字符识别）、表格识别、公式识别和版面分析。

## 主要功能
- **文档解析**：完整的PDF和图像文档解析
- **OCR识别**：文本、公式、表格的单独识别
- **版面分析**：文档结构检测和分析
- **多语言支持**：支持中英文文档
- **批量处理**：支持文件夹批量处理和分组处理

## 技术特点
- 相比MinerU平均提升5.1%性能
- 相比端到端模型，3B参数模型在英文文档上表现最佳
- 多页文档处理速度0.84页/秒
- 支持多种推理后端（LMDeploy、vLLM、transformers、API）
